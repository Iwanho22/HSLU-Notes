Da die [[Integration]] die Umkehrung der [[Ableitung|Differnetiation]] ist ergibt sich auch direkt eine Integrationstabelle

| f(x) | $\int f(x) \, dx$ |
| ---- | ---- |
| $x^{\alpha}$, $\alpha \neq -1$ | $\frac{x^{\alpha+1}}{\alpha+1}+C$ |
| $\cos x$ | $\sin x +C$ |
| $\sin x$ | $-\cos x +C$ |
| $e^{ x }$ | $e^{ x }+C$ |
